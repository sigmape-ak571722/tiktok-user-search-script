## Workflow Steps

### Step 1: Set Up Your MadHub Account

1. Download and install MadHub on your Android device or emulator.

### Step 2: Navigate to TikTok User Search Feature

1. From the main dashboard, select **TikTok User Search**.
2. You will be presented with a user-friendly interface to input your search parameters.

### Step 3: Configure Search Parameters

1. **Keywords**: Enter relevant keywords that represent the interests or profiles of users you want to find.
2. **Filters**:
   - **Follower Count**: Specify the minimum and maximum number of followers to include users in your search.
   - **Following Count**: Set your desired following count range to narrow down the potential users.
   - Additional filters may include account type or post count, depending on your marketing strategy.

### Step 4: Execute the Search

1. Once you have configured the parameters, click on the **Search** button.
2. MadHub will run the search and display a list of TikTok users matching your criteria.

### Step 5: Engage with Selected Users

1. From the results, you can choose to follow users or send them direct messages to initiate engagement.
